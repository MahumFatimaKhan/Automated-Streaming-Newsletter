# Custom Fonts Directory

Place custom fonts here for banner text rendering.

## Recommended Font

For best results, add the following font file:
- `HelveticaforTarget-Bold.ttf` - Helvetica Bold font for banner taglines

## Fallback Fonts

If the custom font is not available, the system will use:
1. Arial Bold (from Windows system fonts)
2. Default system font

## Font Installation

1. Download the Helvetica Bold font file
2. Rename it to `HelveticaforTarget-Bold.ttf`
3. Place it in this directory

The system will automatically detect and use the font when generating banners.
